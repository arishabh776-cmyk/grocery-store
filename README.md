# GlasMart Automatic Grocery Store

A full-stack grocery store demo with a dependency-free Node backend and a glassmorphism frontend.

## Features

- Product catalog with search, category filtering, sorting, stock counts, and real product imagery
- Customer registration and login
- Token-based sessions
- Cart add, update, remove, and totals
- Automatic checkout with tax, delivery fee, stock deduction, and order creation
- Order history
- Admin inventory panel with stock updates and product creation
- Responsive glassmorphism storefront UI

## Run

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000).
